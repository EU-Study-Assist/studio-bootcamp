Introduction to the course

Welcome to the R for Research bootcamp. This program is designed to equip you with the fundamental skills and knowledge needed to leverage the power of R for your research projects. Whether you are a novice with no programming experience experience or an experienced researcher looking to enhance your data analysis capabilities, this bootcamp will provide you with the tools and confidence to effectively use R. Getting Started

Objective

The objective of this module is to ensure that you have all the necessary tools and setup to begin your journey with R. By the end of this module, you will:

  • Have R and RStudio installed and configured on your computer.
  • Be familiar with the RStudio interface and its key features.
  • Understand basic R syntax and commands.

You are expected, to have finished following the instructions below before class starts

Installing R and RStudio

To get started, we need to acquire our copy of R and RStudio. Both software are free to download and use.

Installing R To install R language, you need to download the right software for your operating system.

  • Windows Users: Click on this link and install the downloaded file.

  • Mac Users: The R language software is having 2 different downloads based on your Mac OS processor.

    • For M1-3 Apple silicon processors
    • For older intel macs

  • Linux Users: Open your terminal from applications or use the shortcut ctrl + alt + t to open it anywhere. Download and install R based on your linux distribution:

    • Debian distro: type in or copy the code below:
    #| eval: false
$ sudo apt update
$ sudo apt install r-base r-base-dev
    • Redhat/Fedora distro: type in or copy the code below:
    #| eval: false
$ sudo dnf install R

  • The RStudio Interface: R Studio makes it easier to use R by providing user-friendly environment. The components are:

    • source pane: Located in the top-left part
    • console pane
    • environment/history pane
    • files7plots/packages/help/viewer pane
